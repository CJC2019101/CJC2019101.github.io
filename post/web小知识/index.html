Http Erro Code分类

  • 2XX：表示状态成功。
  • 4XX：客户端错误。
  • 5XX：服务器端错误。

简单的识别爬虫

 User-Agent：用户代理，其实我们的浏览器就是一个用户代理，当我使用浏览器发送请求时，我们的请求头里都会有 User-Agent （用户代理）这个提示信息。我们可以通过这个可以简单的识别"Http请求"是浏览器发送的还是其他程序发送的，但这仅仅是针对哪些一点都不会"Http请求"伪装的小白。
